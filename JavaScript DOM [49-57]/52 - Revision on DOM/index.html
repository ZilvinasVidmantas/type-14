3 užduotis

Sukurkite input (be mygtuko), į kurį įrašius vardą ir nulipus nuo input (t.y. blur event), vardas atsiras po apačia, tvarkingame sąraše (su kableliais): Pvz.: Petras, Jonas, Mantas, Antanas.

4 užduotis

Sukurkite formą su dviem inputais, kurie paims skaičius. Suvedus duomenis - turi alert'int skaičių, kuris arčiausiai 100. Pvz.: fn(101, 81) grąžins 101.
