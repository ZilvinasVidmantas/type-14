    * createLink sukuria nuorodą
    * path dinaminis kelias naudojamas routeryje
  * įgalinkite navigavimą, naudodami useNavigate(), iš pagrindinio puslapio į šį įrašo puslapį
  * Panaudokite useParams(), kad gauti id iš url'objektu
  * Aprašykite ApiService.fetchHouse, pagal id raktą
  * Vieno įrašo puslapyje užkraukite duomenis
  * Pateikite duomenis bet kokiu formatu matomame ekrane - gražiai atvaizduosite - darant namų darbus.
